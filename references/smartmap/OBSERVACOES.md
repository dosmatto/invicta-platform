# Observações sobre o uso do SmartMapPlugin

O SmartMapPlugin é uma referência útil para processamento geoespacial e geoestatística, mas sua lógica segue um fluxo tradicional:

Dados → Interpolação → Zona

O MEAP segue uma filosofia diferente:

Camadas Oficiais → Modelo Matemático → Refinamento → Convergência → Modelo Operacional → Zona Oficial → Novos Dados → Novo Refinamento

Portanto, qualquer aproveitamento deverá respeitar a filosofia evolutiva do MEAP.

A zona de manejo no MEAP não é apenas o resultado de um processamento pontual.

Ela representa conhecimento acumulado, histórico, estabilidade, confiabilidade e maturidade.
