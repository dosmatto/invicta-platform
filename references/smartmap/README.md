# Referência Técnica — SmartMapPlugin

Este diretório contém observações técnicas sobre o repositório SmartMapPlugin, utilizado apenas como referência complementar para o desenvolvimento do MEAP.

Repositório original:
https://github.com/gustavowillam/SmartMapPlugin

## Finalidade

O SmartMapPlugin será utilizado somente para estudo de algoritmos e scripts relacionados a:

- interpolação espacial;
- geoestatística;
- krigagem;
- semivariogramas;
- validação cruzada;
- processamento geoespacial;
- integração com QGIS.

## Restrição

Este repositório não será usado como base arquitetural da Plataforma Invicta.

Toda implementação oficial deverá seguir a arquitetura própria do MEAP.
