# Análise Técnica do SmartMapPlugin

## Objetivo

Registrar quais partes do SmartMapPlugin podem servir como referência para o MEAP.

## Pontos de Interesse

- Algoritmos de interpolação.
- Ajuste de semivariogramas.
- Validação cruzada.
- Comparação entre métodos.
- Geração de mapas interpolados.
- Processamento espacial com Python.
- Uso da API do QGIS.

## Pontos que não serão aproveitados

- Interface.
- Arquitetura.
- Fluxo de uso.
- Organização do plugin.
- Regras finais de negócio.
- Experiência de usuário.
- Estrutura geral do código.

## Diretriz

Todo código estudado deverá ser reinterpretado e adaptado à arquitetura oficial da Plataforma Invicta.
